# AI-Foundations-a-field-guide-to-the-stack
A simple tool for learning how AI works, in plain language. It explains 80 key terms, each with a one-line meaning and a real example. Everything is colour-coded to the "stack" so you see where each piece fits, with diagrams showing how a question flows through AI. Guided trails, search and a progress tracker make it easy to start and build from.
